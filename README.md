Number Guessing Game (Java)

This is a console-based number guessing game built in Java. The user selects a difficulty level and tries to guess a randomly generated number within a limited number of attempts.

This project focuses on clean code structure, input validation, and basic object-oriented programming.

Features
Multiple difficulty levels (Easy, Medium, Hard)
Input validation to prevent crashes and handle invalid entries
Hint system based on how close the guess is
Win/loss tracking using an object-oriented approach
Replay functionality
